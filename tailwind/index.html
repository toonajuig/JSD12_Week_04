มินิมอล + คอมเมนต์ละเอียด + ตัวอย่างฝึกต่อ

หน้าเดียวที่ใช้ เรียน Tailwind ได้ทั้ง layout และ utility classes

ไฟล์นี้ถูกจัดให้เป็นเหมือนชีตสรุปสั้น ๆ โดยแยก section ชัดเจน และใส่คอมเมนต์อธิบาย class เกือบทุกจุด เพื่อให้เปิดดูแล้วอ่านตามได้ทีละบรรทัด

Layout

ฝึก `flex`, `grid`, `gap`, `max-w`

Style

ฝึก `bg-*`, `text-*`, `border`, `shadow`

Responsive

ฝึก `sm:*`, `md:*`, `lg:*`

Quick Summary

หลักคิดของดีไซน์มินิมอล

1. ใช้สีให้น้อย

เมื่อ palette ไม่เยอะ เราจะควบคุมโฟกัสของสายตาได้ง่ายขึ้น

2. ใช้ spacing ให้สม่ำเสมอ

ระยะห่างที่นิ่งและซ้ำกัน ทำให้หน้าเว็บดูเป็นระบบมากขึ้น

3. เน้นลำดับสายตา

ใช้ขนาดตัวอักษร น้ำหนัก และสีอ่อนเข้ม เพื่อบอกว่าอะไรสำคัญก่อน

Responsive Tip

คลาสอย่าง `md:grid-cols-2` หมายถึง เมื่อหน้าจอถึง breakpoint ระดับ `md` ให้ grid เปลี่ยนเป็น 2 คอลัมน์

Colors

ตัวอย่างการใช้สีใน Tailwind

ด้านล่างคือการ์ดตัวอย่างสี เพื่อให้เห็นว่าคลาสกลุ่ม `bg-*`, `text-*`, `border-*` ช่วยกำหนด mood ของหน้าได้ยังไง

`bg-stone-900`

ใช้กับปุ่มหลักหรือจุดที่ต้องการเน้น

`bg-stone-300`

ใช้เป็นสีรองหรือพื้นหลังอ่อน

`bg-white`

ใช้กับ card หลักเพื่อให้คอนทราสต์ชัด

`border-stone-300`

ใช้ตีกรอบเบา ๆ โดยไม่แข็งเกินไป

Buttons

ตัวอย่างปุ่มหลายแบบ

ส่วนนี้ช่วยให้เห็นความต่างของปุ่ม primary, secondary และ ghost พร้อมคลาส `hover:*` และ `transition`

Primary Button

ใช้ `w-full` เพื่อให้ปุ่มกว้างเต็ม card และใช้พื้นเข้มเพื่อเน้น action หลัก

Secondary Button

ปุ่มแบบขอบเหมาะกับ action รอง ที่ควรเห็นแต่ไม่แย่งความเด่นจากปุ่มหลัก

Ghost Button

ไม่มีเส้นขอบและไม่มีพื้นเดิม เหมาะกับ action ที่สำคัญน้อยที่สุด

Cards

ตัวอย่าง card หลายรูปแบบ

ลองดูว่าการเปลี่ยน `padding`, `border`, `background`, `text color` และ `shadow` ช่วยให้ card ให้ความรู้สึกต่างกันยังไง

Simple Card

เรียบที่สุด

ใช้แค่ขอบบาง ๆ และพื้นที่ว่างให้พอ ก็ได้ card ที่สะอาดตาแล้ว

Soft Card

นุ่มกว่าเดิม

เปลี่ยนพื้นหลังให้อ่อนลงแทนการใช้เส้นขอบเยอะ ๆ จะได้ความรู้สึกเบา

High Contrast Card

เน้นจุดสำคัญ

card เข้มเหมาะกับข้อความสำคัญหรือส่วนสรุปที่อยากให้เด่นขึ้นทันที

Responsive

ดูการเปลี่ยน layout ตามขนาดหน้าจอ

ด้านล่างใช้ `grid-cols-1`, `sm:grid-cols-2`, `lg:grid-cols-4` เพื่อให้เห็นว่า breakpoint ทำงานยังไง
